# JetAutoencoder_ntuplizer

This is an ntuplizer to get useful variables for autoencoder training from PF nanoAOD.

## Getting started

For first time:
```shell
git clone https://github.com/ottolau/JetAutoencoder_ntuplizer
cd JetAutoencoder_ntuplizer
source firsttime.sh
```

For later usage:
```shell
source env.sh
```
